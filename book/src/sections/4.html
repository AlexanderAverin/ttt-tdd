¶ Идём к зелёной зоне

Когда мы создали отказной тест, мы оказались в красной зоне. Теперь наша задача — пройти этот тест с как можно меньшими затратами и как можно быстрее.

Что мы можем написать в методе getState, чтобы он проходил тест? Мы можем вернуть начальное состояние игры. Запускаем тест, он пройден. Мы оказались в зелёной зоне.

На первый взгляд кажется, что мы сделали что-то бесполезное. Но мы создали первый метод АПИ, тесты к нему, и уже можем передать его команде. Теперь с указанными методами могут работать другие разработчики.

Перед каждым комитом мы проверяем изменения на написанных тестах, и если что-то сломалось, нам будет проще понять, в какой момент.

Пока что getState всегда возвращает начальное состояние. К этому методу мы вернёмся позже.

test/game.js
  class Game {
    getState() {
-     return null
+     return [
+       ['', '', ''],
+       ['', '', ''],
+       ['', '', '']
+     ]
    }
  }
