¶ Один тест вымещает другой

Теперь посмотрим на код тестов и увидим, что третий тест включает в себя случай, описанный во втором. Поэтому второй тест можно удалить, он нам больше не понадобится.

Это нормальная практика. Код тестов такой же и живой, как продакшн-код, поэтому тесты будут постепенно эволюционировать и какие-то из них будут отмирать.

  it('Sets user\'s symbol in cell with given coordinates', () => {
    const x = 1, y = 1

    game.acceptUserMove(x, y)
    const board = game.getState()
