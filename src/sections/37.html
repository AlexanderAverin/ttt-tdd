¶ Финальные штрихи

Теперь осталось добавить проверку на победу компьютера и ничью, затем написать метод для перезапуска игры. Проверить тесты на запахи и отрефакторить код ещё раз.

Подробно останавливаться на этом сейчас не будем, перейдём к проверке игры в браузере.

Создадим ХТМЛ-страницу с подключением скрипта, из которого мы запустим игру.

Здесь нам пригодится сборка, которую мы настроили ранее. Запускаем в терминале npm run build. Вебпак соберёт игру в один бандл, который можно подключать к странице.

Результат

src/index.js
import DomController from './DomController'
import Game from './Game'

const game = new Game()
const dom = new DomController({
  root: 'body', 
  game
})

dom.init()
